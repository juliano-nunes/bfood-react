## Installation

Use the package manager [npm](https://www.npmjs.com/package/stable/) to install BFood.

```bash
npm install
```

## Usage

#### Init
```bash
REACT_APP_ZOOMATO_API_TOKEN={your zoomato token} npm start
```

#### Run e2e tests
```bash
cypress:open
```
